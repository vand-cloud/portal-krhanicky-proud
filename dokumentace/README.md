# Dokumentace a strategie

Materiály ANFILOV k projektu Krhanický Proud, mimo samotný kód webu: zadání, datové schéma, výzkum pro katalog a strategické dokumenty ke kampani.

## Obsah

- `SPEC.md` — zadání a rozsah projektu
- `SCHEMA.md` — datové schéma katalogu a Úřadu
- `SANITY-MAPPING.md` — mapování polí mezi zdroji a Sanity schématem
- `staging/` — podkladová data z migrace katalogu do Sanity (2026-07-12)
- `vyzkum-cache-katalog/` — uložené výsledky vyhledávání, ze kterých vznikl katalog míst a akcí
- `strategie/` — strategické dokumenty ke kampani Krhanický Proud
- `_IN/` — původní podklady od Ivana (brief, inspirace, brand, obsah, skici)
